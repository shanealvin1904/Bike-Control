Controls

Key	Action
A	Move Left
D	Move Right
W	Mega Jump

Gameplay

Bike Control Game is a physics-based platformer developed in Unity where players navigate through obstacles and challenging terrain using simple but precise controls.

Move left and right using A and D, and use W to perform a powerful Mega Jump to overcome gaps, avoid hazards, and reach the goal. Timing and control are essential, as each level presents new challenges that test the player’s movement skills and reaction speed.

The game focuses on delivering a fun and responsive gameplay experience while exploring Unity’s physics and movement systems.
